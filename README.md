# Gaming-performance-insights
📌 Project Overview

This project explores the relationship between daily gaming habits, sleep quality, and academic/work performance.
The goal is to understand whether gaming has a direct or indirect impact on performance, and how sleep may act as a mediating factor.

🎯 Objective
Analyze whether gaming hours affect performance
Investigate the role of sleep quality in this relationship
Identify patterns between productivity and lifestyle behavior
Support findings using statistical summaries and visual analysis

📊 Dataset Features
Daily Gaming Hours
Sleep Quality Score
Academic/Work Performance Category
Work Productivity Score

🔍 Key Analysis Performed
Boxplot analysis of gaming vs performance
Sleep quality comparison across performance groups
Group-wise statistical averages
Correlation-style interpretation using grouped trends

📊 Key Findings (Data-Driven Results)
Gaming hours show a weak direct relationship with performance
Lower-performing individuals tend to have slightly higher gaming hours
Sleep quality shows a stronger and more consistent relationship with performance
Poor sleep appears more common among lower-performing groups
Gaming may influence performance indirectly through sleep quality

🧠 Final Insight
Gaming alone does not strongly determine performance, but it may contribute indirectly by affecting sleep quality, which has a clearer relationship with academic/work productivity.

📊 Example Group Analysis (Insight Summary)
Higher performance → lower average gaming hours
Lower performance → higher average gaming hours
Better performance → better sleep quality trend
Poor performance → worse sleep quality trend
